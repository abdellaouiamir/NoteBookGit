git branch <branch-name>: Creates a new branch.
git checkout <branch-name>: Switches to an existing branch.
git checkout -b <branch-name>: Creates and switches to a new branch in one step.
git switch <branch-name>: Switches to an existing branch (newer command).
git switch -c <branch-name>: Creates and switches to a new branch (newer command).
